        1. Configuring Git
        git config --global user.name "Your Name"
        git config --global user.email "youremail@example.com"
        
        2. Cloning a Repository
        git clone  "repository_url"
        Example:
        git clone https://github.com/user/repo.git
        
        3. Checking Repository Status
        git status
        
        4. Adding Files to Staging Area
        git add "file_name"         | Stage a specific file
        git add .                   | Stage all changes
        
        5. Committing Changes
        git commit -m "Your commit message"
        
        6. Pushing Changes to Remote Repository
        git push origin "branch_name"
        Example:
        git push origin main
        
        7. Pulling Changes from Remote Repository
        git pull origin "branch_name"
        Example:
        git pull origin main
        
        8. Creating a New Branch
        git branch "branch_name"
        
        9. Switching Branches
        git checkout "branch_name"
        
        10. Merging Branches
        git merge "branch_name"
        
        11. Viewing Commit History
        git log
        
        12. Discarding Changes (Undoing Changes)
        git checkout -- "file_name"
        
        13. Deleting a Branch
        git branch -d "branch_name"

These are the usual Commands
